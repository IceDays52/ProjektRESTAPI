# MC Księgowość – Web Application

A fullstack web application built with React + TypeScript and .NET REST API.

## Tech Stack
Frontend:
- React
- TypeScript
- Vite
- CSS

Backend:
- .NET
- REST API
- Entity Framework
- MySQL

## Features
- Contact form connected to backend API
- Google Maps integration
- Responsive design
- Animated UI

## Future improvements
- Authentication system
- Admin panel
- Better error handling
